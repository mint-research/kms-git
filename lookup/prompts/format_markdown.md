# Prompt
```
"[Hier den Prompt einfügen, dessen Funktion zusammengefasst werden soll.]"
-----
Formatieren Sie den folgenden Text als Markdown, beginnend mit Header 1 (#) für die Hauptüberschrift. Untergliederungen sollen durch Header 2 (##) und, falls notwendig, durch Header 3 (###) organisiert werden. Achten Sie auf eine klare Struktur, indem Sie Absätze logisch trennen, und verwenden Sie Aufzählungen oder hervorgehobene Abschnitte, falls sinnvoll. Der Text soll gut lesbar, übersichtlich und standardmäßig als Markdown formatiert sein. Nummeriere die Überschriften nicht. Verzichte auf Hervorhebung der Überschriften durch Fett- oder Kursiv-Formatierung.
```
